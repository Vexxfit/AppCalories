# VEXX · Nutrición

App de nutrición y entrenamiento (conteo de calorías, macros y tonelaje).
Aplicación web de un solo archivo (`index.html`), sin backend. Los datos se
guardan en el dispositivo del usuario (localStorage); no se sube ninguna
información personal al repositorio.

## Publicar en GitHub Pages
1. Sube estos archivos a la rama `main`.
2. Settings → Pages → Source: **Deploy from a branch** → `main` / `root` → Save.
3. La app quedará en `https://USUARIO.github.io/REPO/`.
